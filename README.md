## NetBook

Just a fun personal project bringing back the absolute glory days of the late-2000s internet. It’s a retro social media profile page (heavily inspired by old-school Facebook) complete with chaotic fake ads and a side-by-side profile layout.

I built this to practice CSS Flexbox and responsive column layouts without making everything look like a generic modern template.

---

## Cool Stuff Inside
*   **3-Column Layout:** Uses Flexbox to cleanly map the navigation, the wall feed, and the right ad sidebar.
*   **Ad Rows:** Figured out how to flex the profile pic and sponsored ads side-by-side.
*   **Hidden Scrollbars:** Keeps the sidebar scrollable for the friends list without displaying the ugly browser scrollbar track.
